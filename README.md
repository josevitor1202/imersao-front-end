# 🚀 Imersão Front-End com IA

Bem-vindo ao meu repositório de estudos da Imersão Front-End! Aqui compartilho a evolução dos meus projetos e a aplicação prática de conceitos de desenvolvimento web.

## 📌 Sobre o Projeto
Este repositório contém o desenvolvimento de interfaces modernas, como o meu catálogo de livros, onde explorei a estruturação com **HTML** e a estilização com **CSS**. O foco é criar páginas responsivas, semânticas e visualmente atraentes.

## 🛠️ Tecnologias Utilizadas
- **HTML**: Estruturação de conteúdo de forma semântica.
- **CSS**: Estilização, layouts e design responsivo.
- **JavaScript**: Implementação de lógica, interatividade e manipulação de dados no Front-end.
- **IA**: Utilizada como assistente de codificação e geração de conteúdo criativo.

## 📊 Meus Interesses e Objetivos
- **Desenvolvimento Front-End**: Criar experiências ricas para o usuário.
- **Ciência de Dados**: Analisar e visualizar informações de forma inteligente.
- **Engenharia**: Aplicar lógica de programação para resolver problemas complexos (visto meu background em Física e Simulação).

Busco aplicar o rigor analítico e a base matemática de Engenharia Física para criar soluções inteligentes e visualmente impactantes em Front-End e Ciência de Dados
